向您介绍 VelaBench for HyperOS

¤什么是 VelaBench
  VelaBench 是首个 Xiaomi/Redmi Vela OS 轻智能穿戴设备平台的性能评估程序，满足网友们赛博斗蛐蛐的爱好。

  我们预计在今年七月左右发布 VelaBench 2 for HyperOS2，将对所有使用 HyperOS 2 的手表手环等设备进行适配。目前开发工作在紧锣密鼓进行中。

  VelaBench 一代版本问世于 2024 年 6 月，首次进行跑分是在 Redmi Watch4 上，开发共进行了两个星期。1.0 版本使用 GPL-3 协议进行开源，使用 VelaBench 源码需标注出处。

¤为什么要开发 VelaBench
  正如前面所说，满足网友赛博斗蛐蛐的爱好，同时也是为了看看自己手上的手表 soc 是个什么水平。

  实际上这个是我硬着头皮读了一下午开发指南，然后一拍脑门就想出来的项目，没什么意义，单纯为了好玩。

¤VelaBench 2 会考虑增加哪些功能
  更好看的 UI，2D 与 3D 图形性能的评估（或许），存储和内存速度评估（或许）以及更精准的跑分算法
